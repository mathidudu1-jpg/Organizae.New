# Organizae.New

Repositório para reconstrução do app Organizae.Space

## Estrutura

- `/public` - Imagens públicas (favicons, logos, etc)
- `/src/assets` - Assets do aplicativo (imagens, ícones)
- `.env` - Variáveis de ambiente

## Setup

```bash
# Instalar dependências
bun install

# Configurar ambiente
cp .env .env.local

# Rodar em desenvolvimento
bun dev
```
